iOS Shortcuts

The following shortcuts work with PetDNS and the iOS Shortcuts app. Tap any title to install the shortcut directly to your device via iCloud. Have a shortcut request? Open an issue on GitHub.

Deny/Allow Domain from Browser

A shortcut that appears in the iOS Share Sheet that will add the domain of the currently open website to the deny or allowlist for a selected profile.

Check NextDNS Connection

Tests your connection to NextDNS and shows what profile the device is using. Can be used inside other Shortcuts to take action based on the result.

AI Overview of Profile

A shortcut that uses Apple's Private Cloud Compute to provide a natural-language overview of the profile's settings. Requires a device compatible with Apple Intelligence.

AI Recommendations for Profile

A shortcut that uses Apple's Private Cloud Compute to provide a natural-language list of recommendations for your profile. Requires a device compatible with Apple Intelligence.
